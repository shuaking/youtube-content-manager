1. **Next.js 16 Breaking Change** - params 现在是 Promise，需要用 `React.use()` 解包
2. **404 错误** - 修复了所有动态路由页面的 params 访问问题
3. **视频播放** - 嵌入真实 YouTube iframe 播放器
4. **构建错误** - 所有 TypeScript 错误已解决

### 使用的技术栈
- Next.js 16 (App Router)
- React 19
- TypeScript (strict mode)
- Tailwind CSS v4
- shadcn/ui 组件
